# National Institutes Schools Portal (NIS) 🏛️

بوابة المعاهد القومية - منصة تعليمية متكاملة تتبع المعاهد القومية في مصر، تهدف إلى تسهيل الوصول لبيانات المدارس، الأخبار، التوظيف، وتوفير أدوات ذكاء اصطناعي متطورة.

## 🚀 التقنيات المستخدمة (Tech Stack)

### **Frontend**
*   **React 19 & TypeScript:** لبناء واجهة مستخدم سريعة وقوية.
*   **Vite:** كأداة تطوير وتجميع سريعة جداً.
*   **Tailwind CSS:** للتصميم العصري والمستجيب (Responsive Design).
*   **Framer Motion:** للأنيميشن واللمسات الجمالية الاحترافية.
*   **React Query (TanStack Query):** لإدارة طلبات البيانات (Data Fetching) والتخزين المؤقت.
*   **Zustand:** لإدارة الحالة العامة للتطبيق (State Management).
*   **Lucide React:** لمجموعة الأيقونات العصرية.
*   **Google Gemini SDK:** لتشغيل ميزات الذكاء الاصطناعي في AI Studio.

### **Backend**
*   **PHP (PDO):** واجهة برمجية (API) آمنة وسريعة.
*   **MySQL:** قاعدة بيانات مستقرة لتخزين كافة بيانات المنصة.
*   **File-Based Caching:** نظام تخزين مؤقت على السيرفر لتقليل زمن الاستجابة (TTFB) لأقل من 5ms.

## 🛠️ كيفية التشغيل (Setup & Installation)

### **المتطلبات:**
*   Node.js (الإصدار 18 أو أحدث)
*   سيرفر محلي (مثل XAMPP أو WampServer) لتشغيل PHP و MySQL.

### **خطوات التشغيل:**
1.  **تهيئ قاعدة البيانات:**
    *   قم بإنشاء قاعدة بيانات جديدة باسم `nis_db`.
    *   قم باستيراد ملف `database.sql` الموجود في المجلد الرئيسي.
2.  **إعداد ملفات Backend:**
    *   ضع محتويات مجلد `backend` في السيرفر الخاص بك.
    *   قم بتعديل بيانات الاتصال في ملف `backend/db_config.php`.
3.  **تشغيل Frontend:**
    ```bash
    # تثبيت المكتبات
    npm install

    # تشغيل المشروع في وضع التطوير
    npm run dev
    ```

## 📊 هيكل قاعدة البيانات (Database Schema)

يعتمد المشروع على تصميم مرن يجمع بين الجداول الثابتة وتخزين الـ JSON في جدول الإعدادات:

1.  **جدول `schools`:** يخزين بيانات المدارس (الاسم، الموقع، المدير، الشعار، والنوع).
2.  **جدول `news`:** للأخبار مع دعم الـ "Featured" والتحكم في النشر.
3.  **جدول `jobs`:** لعرض الفرص الوظيفية المتاحة.
4.  **جدول `settings`:** هو "قلب" المنصة، حيث يتم تخزين البيانات الديناميكية (مثل محتوى صفحة "عن الموقع"، مقدمة الصفحة الرئيسية، طلبات التوظيف، والرسائل المستلمة) بصيغة **JSON**.

## ✨ ميزات برمجية هامة للمطورين

*   **Bilingual System:** دعم كامل للغتين العربية والإنجليزية عبر `LanguageContext`.
*   **Real-time Sync:** يتم تحديث البيانات فورياً في جميع التبويبات المفتوحة عند إرسال شكوى أو طلب توظيف باستخدام `BroadcastChannel`.
*   **Security:** يوجد فحص أمني صارم (Server-side) للملفات المرفوعة في نموذج التوظيف للسماح فقط بملفات `pdf, doc, docx`.
*   **Performance:** يتم تحميل الصور بنظام `Lazy Loading` وتم تطبيق نظام `Cache Busting` لضمان ظهور التعديلات فوراً بعد التحديث من لوحة التحكم.
*   **SEO Ready:** الصفحات الرئيسية وصفحات المدارس والأخبار تدعم Meta Tags ديناميكية لتحسين الظهور في محركات البحث.

## 📂 هيكل المجلدات (Folder Structure)
*   `/src/pages`: تحتوي على جميع صفحات الموقع.
*   `/src/components/common`: المكونات المشتركة (الأزرار، المودال، إلخ).
*   `/src/context`: إدارة الحالة واللغة والبيانات.
*   `/src/services`: الاتصال بالـ API وخدمات Gemini.
*   `/backend`: تحتوي على `api.php` والشيفرات البرمجية للسيرفر.
